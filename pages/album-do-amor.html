Álbum do Amor

Uma experiência digital personalizada que simula um álbum de figurinhas. Conta com sistema de login, pacotinhos com animações WebGL e gestão de inventário em tempo real.

Tipo Projeto Pessoal
Stack Principal React, GSAP, Firebase
Status Concluído (Mai/2026)

Sobre o projeto

O Álbum do Amor foi idealizado como um presente digital imersivo. O usuário faz login no sistema e ganha acesso a pacotinhos virtuais que, ao serem abertos, revelam figurinhas animadas que vão preenchendo as páginas de um álbum virtual de memórias.

Objetivo

Criar uma experiência gamificada e emocional. O grande desafio técnico foi garantir que o sistema de inventário funcionasse em tempo real e que as animações de rasgar o pacote parecessem o mais fluidas e naturais possíveis no navegador.

Features Destaque

  • Animação WebGL de rasgar pacotinho de figurinha.
  • Autenticação e gestão de banco de dados via Firebase Firestore.
  • Fundo interativo de corações utilizando CSS Gooey e GSAP.
  • Trilha sonora ambiente integrada à interação.

Tecnologias

  • HTML5 / CSS3
  • React JS
  • GSAP (Animações)
  • Firebase Firestore
  • Vite
