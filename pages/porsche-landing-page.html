Porsche Landing Page

Landing page premium inspirada na Porsche, desenvolvida com GSAP e Swiper.js para proporcionar uma experiência visual cinematográfica e altamente interativa.

Tipo Projeto de Estudo
Stack Principal JavaScript, GSAP e Swiper.js
Status Concluído (Mar/2026)

Sobre o projeto

Porsche Landing Page é uma experiência digital inspirada na identidade visual da Porsche, desenvolvida para apresentar diferentes modelos da marca através de uma interface elegante, imersiva e cinematográfica. O projeto foi criado com foco em animações avançadas, transições fluidas e uma navegação moderna, proporcionando uma experiência premium semelhante às campanhas digitais de grandes montadoras. Mais do que uma landing page institucional, o projeto explora técnicas modernas de UI/UX para criar impacto visual e destacar os veículos de forma dinâmica e envolvente.

Objetivo

O principal objetivo foi aprofundar conhecimentos em animações avançadas para web, manipulação de elementos do DOM e arquitetura modular em JavaScript. O projeto busca reproduzir a sensação de exclusividade e sofisticação associada à marca Porsche através de uma experiência visual refinada e altamente responsiva.

Features Destaque

  • Animações avançadas com GSAP.
  • Navegação vertical entre veículos com Swiper.js.
  • Efeito Coverflow com profundidade e transições suaves.
  • Layout totalmente responsivo.
  • Estrutura modular em JavaScript Vanilla.
  • Design inspirado na identidade visual da Porsche.
  • Tipografia personalizada e foco em experiência premium.

Tecnologias

  • HTML5
  • CSS3
  • JavaScript
  • GSAP
  • Swiper.js
